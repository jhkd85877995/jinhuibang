# 锦汇邦本地生活服务平台 - 设计规格说明

## 项目概述
- **项目名称**：锦汇邦本地生活服务平台
- **品牌标志**：锦汇邦
- **目标**：1:1 复刻米宿管家房东管理系统
- **技术栈**：CloudBase + 微信小程序 + PC 管理后台

---

## DESIGN SPECIFICATION

### 1. Purpose Statement
锦汇邦本地生活服务平台旨在为本地生活服务提供一个综合性的管理工具，包含租房管理、本地同城服务、共享扫码三大核心模块。平台面向房东、服务商和管理员，提供高效的资源管理、订单处理、财务统计等功能。界面需要体现专业、可信赖的企业级风格，同时保持操作的高效性和直观性。

### 2. Aesthetic Direction
**选择：Industrial/utilitarian（工业/实用主义）**

理由：
- 符合企业管理后台的专业定位
- 强调功能性和效率，避免过度装饰
- 参考米宿管家的实用风格，但提升视觉质感
- 使用清晰的视觉层次和功能性设计元素

### 3. Color Palette
- **主色**：`#1E3A8A`（深蓝 - 专业、可信赖）
- **辅助色**：`#3B82F6`（亮蓝 - 交互元素、链接）
- **强调色**：`#F59E0B`（琥珀色 - 重要提示、数据统计）
- **背景色**：`#F9FAFB`（浅灰 - 页面背景）
- **文字色**：`#111827`（深灰 - 正文）、`#6B7280`（中灰 - 辅助文字）
- **边框色**：`#E5E7EB`（浅灰 - 分割线、边框）

**禁止使用**：紫色、紫罗兰、靛蓝、紫红色及其渐变

### 4. Typography
- **标题字体**：`PingFang SC`（中文字体）、`SF Pro Display`（英文）
- **正文字体**：`PingFang SC`（中文字体）、`SF Pro Text`（英文）
- **等宽字体**：`SF Mono`（用于数据展示）
- **禁止使用**：Inter、Roboto、Arial、Helvetica、system-ui、-apple-system

### 5. Layout Strategy
- **整体布局**：经典的企业管理后台布局
  - 顶部：导航栏（Logo + 主导航 + 用户信息）
  - 左侧：多级菜单（支持 1-4 级展开）
  - 主内容区：卡片式布局 + 数据表格
  
- **设计特点**：
  - 使用清晰的视觉分割，避免元素拥挤
  - 数据表格采用斑马纹提升可读性
  - 卡片设计带有轻微阴影和圆角
  - 保持对齐和间距的一致性
  
- **打破常规**：
  - 仪表盘使用不对称的数据卡片布局
  - 关键数据使用大字号 + 强调色突出
  - 使用图标 + 文字的导航设计

## 平台区分

### 微信小程序（房东端）
- 简化版的管理功能
- 重点：房源查看、订单处理、消息通知
- 风格：保持与 PC 端一致的色彩和字体，但布局适配移动端

### PC 超级管理后台
- 完整的管理功能（对应您发的截图）
- 重点：数据统计、资源配置、系统管理
- 风格：Industrial/utilitarian（工业/实用主义）

## 技术实现要点

### CloudBase 集成
- **认证**：微信小程序使用 OPENID 自动登录，PC 端使用 CloudBase Auth（账号密码）
- **数据库**：使用 CloudBase NoSQL 文档数据库
- **存储**：使用 CloudBase 云存储（图片、文件）
- **云函数**：处理复杂业务逻辑

### 前端技术栈
- **小程序**：原生微信小程序开发（使用 `wx.cloud`）
- **PC 后台**：建议使用 React + Tailwind CSS（或 Vue + Element UI）
